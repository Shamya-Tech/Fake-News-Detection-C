# Fake News Detection (C)

A lightweight Fake News Detection MVP built using the C programming language.
The system classifies user-provided news text as REAL or FAKE using text similarity analysis.

## Features
- Command-line based input
- Text similarity-based classification
- Clear and instant output

## Tech Stack
- C Programming Language
- GCC Compiler
- Visual Studio Code (VS Code)

## MVP
A terminal-based application that accepts a news sentence and displays a classification result.
